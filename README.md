# Business Sales Performance Analytics Dashboard

## Project Overview

This project was completed as part of the Future Interns Data Science & Analytics Internship Task 1.

The objective of this project is to analyze business sales data and generate meaningful insights that can help businesses improve decision-making and identify growth opportunities.

## Tools Used

* Power BI
* Microsoft Excel
* Superstore Dataset

## Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Profit Margin %

## Dashboard Features

* Sales Trend Analysis
* Category-wise Sales Performance
* Region-wise Sales Performance
* Segment Analysis
* Interactive Filters (Region, Category, Year)

## Key Insights

* Technology category generated the highest sales revenue.
* West region contributed the highest overall sales.
* Sales showed consistent growth over the observed period.
* Consumer segment contributed significantly to total sales.
* Overall profit margin was 12.47%.

## Business Recommendations

* Increase focus on high-performing Technology products.
* Expand marketing efforts in the West region.
* Improve sales strategies in lower-performing regions.
* Strengthen customer retention programs for Consumer segment customers.
* Continue monitoring profit margins to improve profitability.

## Outcome

This dashboard helps business stakeholders monitor performance, identify trends, and make data-driven decisions for future growth.
